Python Portable - CRM System

Esta pasta é destinada para uma versão portátil do Python.
Atualmente, o sistema baixa e instala Python automaticamente.
 
Para usar Python portátil:
1. Baixe Python portátil de: https://www.python.org/downloads/windows/
2. Extraia nesta pasta
3. O instalador usará esta versão em vez de baixar automaticamente
