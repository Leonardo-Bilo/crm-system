# Como rodar o backend do CRM

1. Clique duas vezes no arquivo run_backend.bat
2. Aguarde a instalação das dependências (só na primeira vez)
3. O backend vai iniciar e ficará disponível em http://localhost:5000
4. Agora você pode abrir o aplicativo CRM Desktop normalmente!

Se aparecer alguma mensagem de erro, envie um print para o suporte.

## Requisitos
- Python 3.8 ou superior instalado no computador
