======================================== 
   CRM SYSTEM - INSTRUÇÕES DE INSTALAÇÃO 
======================================== 
 
PRÉ-REQUISITOS: 
- Python 3.8 ou superior instalado 
- Windows 10/11 
 
INSTRUÇÕES: 
1. Extraia todos os arquivos desta pasta 
2. Execute o arquivo "instalar_crm_completo.bat" 
3. Aguarde a instalação automática 
4. Use o atalho "CRM System" na área de trabalho 
 
OBSERVAÇÕES: 
- A instalação não requer privilégios de administrador 
- O sistema será instalado na área de trabalho 
- Cada usuário terá seu próprio banco de dados 
 
SUPORTE: 
Em caso de problemas, verifique se o Python está instalado 
e se o antivírus não está bloqueando a instalação.
